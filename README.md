# NER-Transformer-Project
Named Entity Recognition using Transformers

This project implements a Named Entity Recognition (NER) system using HuggingFace Transformers.

Features
Data preprocessing and tokenization
Transformer-based NER model
Training using HuggingFace Trainer API
Evaluation with standard NER metrics

Technologies Used
Python
HuggingFace Transformers
PyTorch
Datasets Library
